Bài Học Nâng Cao Về Java: Lambda Expressions

  • 29/12/2024
  • 3-minute read

Giới thiệu về Lambda Expressions

Lambda Expressions (biểu thức Lambda) được giới thiệu trong Java 8, cho phép viết code theo phong cách lập trình hàm (functional programming) một cách ngắn gọn và biểu cảm. Lambda Expressions có thể được coi là một hàm ẩn danh (anonymous function).

Cú Pháp

  • (tham_số): Danh sách các tham số của hàm. Có thể bỏ trống nếu không có tham số.
  • ->: Toán tử Lambda.
  • biểu_thức_hoặc_khối_lệnh: Phần thân của hàm. Nếu chỉ có một biểu thức, có thể bỏ qua dấu ngoặc nhọn {} và từ khóa return. Nếu có nhiều câu lệnh, cần sử dụng dấu ngoặc nhọn {} và từ khóa return (nếu hàm trả về giá trị).

Functional Interfaces

Lambda Expressions chỉ có thể được sử dụng với Functional Interfaces. Functional Interface là interface chỉ có một phương thức trừu tượng (abstract method). Annotation @FunctionalInterface có thể được sử dụng để đánh dấu một interface là Functional Interface (trình biên dịch sẽ báo lỗi nếu interface không thỏa mãn điều kiện).

Một số Functional Interfaces được định nghĩa sẵn trong Java:

  • Predicate<T>: Nhận một tham số kiểu T và trả về giá trị boolean.
  • Consumer<T>: Nhận một tham số kiểu T và không trả về giá trị.
  • Function<T, R>: Nhận một tham số kiểu T và trả về giá trị kiểu R.
  • Supplier<T>: Không nhận tham số và trả về giá trị kiểu T.

Ví dụ minh họa chi tiết

import java.util.Arrays;
import java.util.List;
import java.util.function.Consumer;
import java.util.function.Function;
import java.util.function.Predicate;

public class LambdaExample {
    public static void main(String[] args) {
        // Lambda expression đơn giản
        Greeting greeting = message -> System.out.println("Xin chào: " + message);
        greeting.greet("Thế giới!");

        // Lambda expression với nhiều tham số
        Calculator calculator = (a, b) -> a + b;
        int result = calculator.calculate(5, 3);
        System.out.println("Kết quả: 5 + 3 = " + result);

        // Sử dụng Predicate
        Predicate<Integer> isEven = n -> n % 2 == 0;
        System.out.println("4 là số chẵn? " + isEven.test(4));
        System.out.println("5 là số chẵn? " + isEven.test(5));

        // Sử dụng Consumer
        List<String> names = Arrays.asList("Alice", "Bob", "Charlie");
        Consumer<String> printName = name -> System.out.println("Tên: " + name);
        names.forEach(printName); // Sử dụng forEach với Lambda

        //Sử dụng Function
        Function<String, Integer> stringLength = s -> s.length();
        System.out.println("Độ dài chuỗi 'Hello': " + stringLength.apply("Hello"));

        //Lambda với khối lệnh
        Calculator calculator2 = (a, b) -> {
            int sum = a + b;
            return sum * 2;
        };
        System.out.println("Kết quả (khối lệnh): 5 + 3 * 2= " + calculator2.calculate(5, 3));
    }
}

@FunctionalInterface
interface Greeting {
    void greet(String message);
}

@FunctionalInterface
interface Calculator {
    int calculate(int a, int b);
}
